# TemuAksi

TemuAksi adalah aplikasi berbasis web yang dikembangkan untuk mempertemukan **penyelenggara acara** dengan **perusahaan/sponsor** dalam satu platform digital. Sistem ini memudahkan proses pengajuan proposal sponsorship, pencarian event, hingga pengelolaan kerja sama secara lebih efektif dan terstruktur.

## ✨ Fitur Utama

### 🔐 Autentikasi & Hak Akses
- Login dan Register
- Multi-role user:
  - Admin
  - Penyelenggara Acara
  - Perusahaan / Sponsor

### 👨‍💼 Admin
- Dashboard admin
- Kelola data pengguna
- Kelola data event
- Monitoring proposal sponsorship
- Laporan data sistem

### 🎉 Penyelenggara Acara
- Membuat dan mengelola event
- Mengirim proposal sponsorship
- Melihat status proposal
- Mencari sponsor potensial

### 🏢 Perusahaan / Sponsor
- Melihat proposal masuk
- Menerima / menolak proposal
- Memberikan pendanaan sponsorship
- Melihat daftar kerja sama event

## 🛠️ Teknologi yang Digunakan

- Laravel 10
- PHP
- MySQL
- Blade Template
- Bootstrap / CSS

## 🎯 Tujuan Project

Project ini dibuat untuk membantu proses kerja sama sponsorship antara penyelenggara acara dan perusahaan agar lebih cepat, transparan, dan efisien dibandingkan proses manual.

## 👥 Project Kelompok

Project ini dibuat sebagai tugas/project kelompok untuk memenuhi kebutuhan pengembangan sistem informasi berbasis web.
